# Installation from source


First, download the project locally:
```shell
git clone git@github.com:MartinBraquet/task-allocation-auctions.git
cd task-allocation-auctions
```

#### Environment

If not already done, create a virtual environment using your favorite environment manager. For instance using conda:

```shell
conda create -n gcaa python=3.12
conda activate gcaa
```

#### Main Installation

Install the package and its dependencies in editable mode:

```shell
pip install -e .
```
